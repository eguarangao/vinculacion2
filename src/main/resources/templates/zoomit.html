Software para lectura

Las personas con discapacidad visual utilizan los amplificadores de pantalla para ampliar la información presentada en la pantalla de un sistema informático. La característica clave de su funcionamiento es que amplía la información de la pantalla independientemente de las aplicaciones que se estén ejecutando. Por lo tanto, un amplificador de pantalla tiene que interactuar con el sistema operativo y las aplicaciones de una computadora para realizar tres tareas principales:

  1. Capturar la información que se presentan en la pantalla.
  2. Determinar el área que se ampliará para mostrar al usuario.
  3. Presentar la información ampliada en la pantalla.

ZoomIt es un magnificador de pantalla (o sistema de ampliación de pantalla), anotación y grabación de pantalla para presentaciones técnicas. ZoomIt se ejecuta en segundo plano y se activa con atajos de teclado personalizables para acercar un área de la pantalla, moverse mientras se acerca y dibujar en la imagen ampliada. ZoomIt funciona en todas las versiones de Windows

En la pantalla principal de ZoomIt (Figura 1) se puede configurar los atajos de teclado, el nivel de zoom que el usuario desea o, simplemente, se mantiene la configuración por defecto y se hace clic en “Ok”. A continuación, se explicarán las funciones más importantes que ayudarán a personas con discapacidad visual al momento de interactuar con una computadora.

Ampliar la pantalla

Si se dejó la configuración de atajos que viene por defecto y ya con el programa ejecutándose, se procede a pulsar la combinación de teclas Ctrl + 1 para ampliar la pantalla. Se puede controlar el zoom de la pantalla. Si se desea ampliar más la pantalla, simplemente se desliza la rueda del ratón para arriba o se pulsa la tecla de flecha para arriba. Si se desea alejar la pantalla, se desliza la rueda del ratón para abajo o se pulsa la tecla de flecha para abajo. En la Figura 2 se muestra a la izquierda la pantalla normal sin amplificar, mientras que a la derecha se muestra la pantalla amplificada con ZoomIt. Para salir de este modo simplemente se presiona la tecla Esc.

Dibujos y anotaciones en pantalla

Si se desea empezar a dibujar o a realizar anotaciones mientras está ampliada la pantalla, simplemente se presiona el botón izquierdo del ratón y para desactivar esta función se pulsa el botón derecho del ratón. También se puede acceder a esta función con la combinación de teclas Ctrl + 2, siempre y cuando no se haya ampliado la pantalla. Para saber si esta función está activa, el puntero del ratón cambia a una cruz como se muestra en la Figura 3.
